<...

HTML шпаргалка.

Главные теги

  • <!DOCTYPE> - Версия HTML документа
  • <html> - html документ
  • <body> - Тело документа

Комментарий

  • <!-- Коммен­тарий не будет отобра­жаться пользо­вателю -->

Формы

  • <form> - Форма
  • <fieldset> - Группи­ровка полей
  • <legend> - Заголовок формы
  • <label> - Название ввода
  • <input> - Ввод
  • <select> - Список
  • <optgroup> - Группа внутри списка
  • <option> - Элемент списка
  • <textarea> - Ввод длинного текста
  • <button> - Кнопка

Текст

  • <h1> до <h6> - Заголовки
  • <p> - Параграф
  • <br> - Перевод строки
  • <strong> - Жирный текст
  • <em> - Курсивный текст
  • <pre> - Не форматированый текст
  • <mark> - Отмеченый текст
  • <del> - Зачеркнутый текст
  • <ins> - Подчеркнутый текст
  • <sup> - Над текст
  • <sub> - Под текст

Цитата и Аббревиатуры

  • <q> - Одно строчная цитата
  • <blockquote> - Блочное цитата
  • <abbr> - Аббревиатура

Списки

  • <ol> - Нумерованный список
  • <ul> - Не нумерованный список
  • <li> - Элемент списка
  • <dl> - Список с описанием
  • <dt> - Термин списка с описанием
  • <dd> - Описание списка с описанием

Ссылки

  • <a href="http://yandex.ru"> - Ссылка на внешнюю страницу
  • <a href="page2"> - Ссылка на внутреннюю страницу
  • <a href="mailto:"> - Ссылка на почту
  • <a href="name"> - Якорь
  • <a href="#name"> - Ссылка на якорь
  • <a href="url" target="_blank> - Открыть в новом окне

Изображения

  • <img src="путь к картинке"> - Изоброжение
  • <map> - Карта изображения
  • <area> - Облость карты изображения

Таблица

  • <table> - Таблица
  • <caption> - Надпись таблицы
  • <thead> - Заголовочная часть таблицы
  • <tfoot> - Нижняя часть таблицы
  • <tbody> - Тело таблицы
  • <colgroup> - Группировка колонок
  • <col> - Колонка
  • <th> - Заголовок
  • <tr> - Строка
  • <td> - Столбец
